Ubuntu系统中常见的命令

process: 0

  1. Ubuntu系统中常见的命令
    1. 系统信息
      1. ls - 列出目录中的文件和文件夹。
      2. pwd - 显示当前工作目录。
      3. whoami - 显示当前登录的用户名。
      4. date - 显示当前日期和时间。
    2. 文件操作
      1. cd [目录名] - 更改当前目录。
      2. cp [源文件/目录] [目标文件/目录] - 复制文件或目录。
      3. mv [源文件/目录] [目标文件/目录] - 移动或重命名文件或目录。
      4. rm [文件/目录] - 删除文件或目录。
      5. mkdir [目录名] - 创建一个新目录。
      6. rmdir [目录名] - 删除一个空目录。
    3. 文件内容操作
      1. cat [文件名] - 查看文件内容。
      2. more [文件名] - 分页显示文件内容。
      3. less [文件名] - 类似more，但允许反向导航。
      4. tail [文件名] - 查看文件的最后几行。
      5. head [文件名] - 查看文件的前几行。
    4. 权限和所有权
      1. chmod [权限] [文件/目录] - 更改文件或目录的权限。
      2. chown [用户:组] [文件/目录] - 更改文件或目录的所有者和组。
    5. 软件包管理
      1. apt-get update - 更新可用软件包列表。
      2. apt-get install [软件包名] - 安装软件包。
      3. apt-get remove [软件包名] - 移除软件包。
      4. apt-get upgrade - 升级所有可升级的软件包。
      5. apt-get autoremove - 自动移除不再需要的软件包。
    6. 系统管理
      1. reboot - 重启系统。
      2. shutdown -h now - 立即关机。
      3. df -h - 显示磁盘使用情况。
      4. free -m - 显示内存使用情况。
      5. top - 显示系统运行的进程。
    7. 网络管理
      1. ping [网址/地址] - 检查网络连接。
      2. ifconfig - 显示或配置网络接口。
      3. netstat - 显示网络连接、路由表、接口统计等。
      4. wget [URL] - 从网络上下载文件。
