LED闪烁实验

实验现象：下载程序后“LED模块”的D1指示灯闪烁

代码：

#include "reg52.h"
typedef unsigned int u16;	//对系统默认数据类型进行重定义
typedef unsigned char u8;
sbit LED1=P2^0;             //将P2.0管脚定义为LED1
void delay_10us(u16 t)
{
  while(t--);	
}
void main()
{	
  while(1)
  {
    LED1=0;	//点亮
    delay_10us(50000); //大约延时450ms
    LED1=1;	//熄灭
    delay_10us(50000); 	
  }		
}

  • 函数功能 : 延时函数，t=1时，大约延时10us

  • 函 数 名 : main
